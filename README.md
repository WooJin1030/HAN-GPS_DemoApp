# GPS Demo App

### 사용자의 위치 정보를 서버로 전송하여 웹 지도로 표시 가능하게 하는 안드로이드 어플

### [leaflet 지도 웹](https://github.com/WooJin1030/HAN-GPS_DemoLeaflet)

## 기술

- React Native

## 앱의 기능

- 회원가입
- 로그인, 로그아웃
- 정해진 시간 간격으로 서버로 위치 정보 보냄
- 자신의 위치가 범위 안인지 밖인지 확인 가능
